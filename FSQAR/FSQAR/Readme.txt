
=======================================================
The Flight Simulator Quick Access Recorder and Analyzer
=======================================================

Version: 1.0
Date: January 9, 2016


This is a Visual Studio 2013 solution that includes all needed to build the application. 
It was created and examined for Prepar3d version 3.1 and SDK version 3.1.2.15831 (actually at January 2016). 
To build the application just load the solution in Visual Studio and press 'Build', that is all.

See Readme.htm for details.
